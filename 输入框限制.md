# 数字输入框要求
## 1. 卷（卷容量：__ GB）
### 1.1 创建卷容量配置要求
#### 1) 范围限制（业务校验）

- 最小值：**10 GB**
    - 合法示例：`10`、`10.0`、`10.00` 
- 最大值：**实际存储池可分配容量**（显示/计算得到的 max 值）
    - 合法示例：`max`、`max.0`、`max.00`（若max是整数）
> 判定规则：输入值 **必须满足** `10 ≤ value ≤ max`
#### 2) 格式限制（前端输入/提交校验）
**2.1 禁止前导 0（多位数）**
- 非法：`010`、`00010`、`0010.0`
**2.2 禁止“有小数点但没有小数”**
- 非法：`10.`、`0010.`、`0.`
**2.3 小数位数仅允许 0~2 位**
- 非法：`100.233`
**2.4 仅允许数字 + 1 个小数点**
- 非法：`10..`、 `10`、`10` 、`10,5`、`1e3`、`10GB`、中文字符、逗号分隔、其他特殊符号
